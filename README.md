# FoodTracker Telegram Bot

FoodTracker — это телеграм-бот для отслеживания ежедневного потребления воды, калорий, физической активности и контроля прогресса. Бот помогает формировать здоровые привычки, рассчитывает индивидуальные нормы и ведёт статистику по дням.

## Возможности

- Настройка профиля пользователя (вес, рост, возраст, активность, город)
- Автоматический расчёт дневной нормы воды и калорий с учётом погоды
- Логирование выпитой воды, приёмов пищи и тренировок
- Получение информации о продуктах через OpenFoodFacts API
- Учёт сожжённых калорий и дополнительной воды за тренировки
- Просмотр ежедневного прогресса и статистики

## Быстрый старт

1. **Клонируйте репозиторий:**
	```bash
	git clone https://github.com/yourusername/foodtracker.git
	cd foodtracker
	```

2. **Создайте и настройте .env файл:**
   
	Пример содержимого:
	```env
	TOKEN=ваш_telegram_bot_token
	OPENWEATHER_API_KEY=ваш_openweathermap_api_key
	FATSECRET_ACCESS_TOKEN=ваш_fatsecret_access_token (если используется)
	```

3. **Установите зависимости:**
	```bash
	pip install -r requirements.txt
	```

4. **Запустите бота:**
	```bash
	python main.py
	```

	Или с помощью Docker:
	```bash
	docker build -t foodtracker .
	docker run --env-file .env -p 8000:8000 foodtracker
	```

## Основные команды бота

- `/set_profile` — настройка профиля (вес, рост, возраст, активность, город)
- `/log_water <мл>` — записать количество выпитой воды
- `/log_food <название продукта>` — добавить приём пищи
- `/log_workout <тип> <минуты>` — добавить тренировку
- `/check_progress` — посмотреть прогресс за день

## Технологии

- Python 3.12
- [aiogram](https://docs.aiogram.dev/) — Telegram Bot API
- SQLAlchemy (async)
- SQLite (по умолчанию)
- Docker

## Структура проекта

- `main.py` — точка входа, запуск бота
- `database/` — модели, инициализация и утилиты работы с БД
- `routers/` — обработчики команд и логики бота
- `middlewares/` — промежуточные слои (например, для сессии БД)

## Переменные окружения

- `TOKEN` — токен Telegram-бота
- `OPENWEATHER_API_KEY` — API-ключ OpenWeatherMap для получения погоды
